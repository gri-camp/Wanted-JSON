## 2. Бесплатный REST API

Предоставляет доступ к открытым данным в формате JSON через RESTful‑эндпоинты. Подходит для прототипирования, обучения и тестирования.


### Основные возможности:

* **Многообразие эндпоинтов** с данными разной тематики:
    * книги (`/books`);
    * спортсмены (`/athletes`);
    * видеоигры (`/videoGames`);
    * фильмы (`/movies`);
    * автомобили (`/cars`) и др.

* **Поддержка основных HTTP‑методов (CRUD):**
    * `GET` — получить данные;
    * `POST` — добавить новую сущность;
    * `DELETE` — удалить сущность по ID;
    * `PATCH` — обновить сущность по ID.

* **Настраиваемые запросы через query‑параметры:**
    * фильтрация (`?genre=фэнтези`);
    * сортировка (`?sort=pageCount:desc`);
    * пагинация (`?page=2&limit=10`);
    * поиск (`?q=матрица`);
    * выборка, проекция полей (`?select=title,author,pageCount`) 

* **Моковая авторизация:**
    * вход через `/fakeAuth/signin`;
    * регистрация через `/fakeAuth/signup`.

### Особенности:      

* **Обработка ошибок** с визуализацией невалидных параметров запроса.
* **Высокая нагрузочная способность** — сервис выдерживает высокие нагрузки 24/7.
* **Старт без оплаты** — бесконечный доступ к API после регистрации (есть суточный лимит запросов).
* **Настройка данных** — поддержка query-параметров для сортировки, пагинации, фильтрации, лимитрования, поиска и выборки данных.
* **Тесты на проверку знаний** — тесты для проверки знаний по разным технологиям.
* **Регистрация и авторизация** — для доступа эндпоинтам, обучающим тестам и модулю внутренний статистики

### Кому подойдёт:
* начинающим разработчикам — для изучения основ работы с API и JSON;
* тестировщикам — для проверки бэкенда или фронтенда;
* разработчикам — для быстрого прототипирования без настройки собственного бэкенда;
* студентам и преподавателям — для учебных проектов и демонстрации работы с REST API.
