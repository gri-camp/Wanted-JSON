Документация для работы с REST API.

Ниже описаны схемы запросов rocket_launch, ответов data_object и ошибок error по эндпоинту: . Пример кода для запроса представлен на JavaScript, но Вы можете использовать любой язык для обращения к сервису по протоколу HTTP.


Каждый пользователь получает бесплатный суточный лимит в запросов, доступный после авторизации и получения API-ключа:

  1. Пройди регистрацию;
  2. Авторизуйся на сайте и перейди в личный профиль;
  3. Скопируй токен доступа вида: eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOjQ3... на странице профиля;
  4. Добавь токен в настройки метода выполнения HTTP-запросов - ниже показаны примеры с Fetch API.

Для увеличения дневного лимита запроса - обратись в техническую поддержку, написав по адресу: mail@wantedjson.ru
